# Windows-DNS-Checker

<div dir="rtl">
**Windows DNS Checker** یک ابزار ساده نوشته‌شده با **PowerShell** است که
بررسی می‌کند آیا یک دامنه از طریق چندین **DNS Server** مختلف قابل
**Resolve** شدن است یا خیر.

این ابزار چندین DNS را به‌صورت **همزمان (Parallel)** تست می‌کند و نتیجه را
به شکل **OK / FAIL** به همراه **IP Resolve شده** نمایش می‌دهد.

توسعه داده شده توسط **Ali Nezamifar**\
قدرت گرفته از **Bia2Host.Com**
</div>

------------------------------------------------------------------------

## ویژگی‌ها

<div dir="rtl">
-   بررسی Resolve شدن دامنه از طریق چندین DNS
-   تست DNSها به‌صورت **همزمان (Parallel)** برای سرعت بیشتر
-   نمایش **IP Resolve شده**
-   خروجی رنگی
    -   سبز = OK\
    -   قرمز = FAIL
-   سازگار با **Windows PowerShell**
-   سبک و فقط شامل یک اسکریپت
</div>

## نحوه اجرا

<div dir="rtl">
برای اجرای اسکریپت دستور زیر را در **PowerShell** اجرا کنید:
</div>

``` powershell
powershell -ExecutionPolicy Bypass -File .\Windows-DNS-Checker.ps1
```

<div dir="rtl">
یا از طریق **Run ویندوز** اجرا کنید:
</div>

``` text
powershell -ExecutionPolicy Bypass -File "C:\path\to\Windows-DNS-Checker.ps1"
```
